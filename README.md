# 📝 Task Manager

A simple and clean Task Manager app built using **Flutter**. This app allows users to add, view, and delete tasks easily.

## 🚀 Features

- Add new tasks
- View a list of added tasks
- Delete tasks with a single tap
- Clean and minimal UI
